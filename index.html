Project 1

For my web design project, I developed a simple yet visually appealing website for yoga classes using HTML and CSS. The site features a clean and calming design that reflects the essence of yoga. I included a navigation bar for easy access to different sections, such as class schedules, instructor details, and contact information. The homepage introduces the benefits of yoga and provides a welcoming message. I incorporated high-quality images to enhance the visual appeal and used a soothing color palette. The website is fully responsive, ensuring accessibility across different devices. Through this project, I focused on creating an intuitive user experience while improving my skills in front-end web development.
